> ###  Academic, integrated

Gnostix 是一个面向学生和老师的信息整合软件开与发与维护此软件的同名组织所共有的名称。

`The Gnostix Project` Github 团队账号主要用于存放第三方扩展开发所需的脚手架/实例代码及其他信息。
